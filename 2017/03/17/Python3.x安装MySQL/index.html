Python3.x安装MySQL-Python

发表于 2017-03-17 | 分类于 Python | 游览 次

今天在学Python爬虫,使用BeautifulSoup4+Requests

使用Requests获取到网页,再通过BeautifulSoup对其中的标签进行查找

接着要将数据插入数据库中,在网上查找的资料是使用mysql-python

由于自己使用的Python版本是3.x,所以不支持pip install mysql-python

最后解决方法是下面这个:

mysql-python 只支持2.x,不支持python 3.x如果是python 3.x，用 pip install MySQLClient

同时发现在Python3.x中,可以直接import pymysql

使用时:conn = pymysql.connect(“host”, “user”, “passwd”, “db_Name”)链接数据库就行了
