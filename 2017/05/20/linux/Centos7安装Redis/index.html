Centos7 安装Redis

发表于 2017-05-20 | 分类于 GEEK ， 服务器 | 游览 次

由于项目中有使用到Redis这个key-value数据库,于是在自己服务器上面进行安装进行学习.
简单介绍一下:
Redis是一个开源的使用ANSI C语言编写、支持网络、可基于内存亦可持久化的日志型、Key-Value数据库，并提供多种语言的API。
选择你需要下载的redis版本,个人安装的是redis-3.2.8这个版本
同时在编译之前需要安装gcc
安装步骤挺简单的,首先先下载对应的版本,然后进行解压,进入根目录然后进行进行编译

[root@VM_52_146_centos ~]#yum install gcc-c++
[root@VM_52_146_centos ~]# cd /home
[root@VM_52_146_centos home]# wget http://download.redis.io/releases/redis-3.2.8.tar.gz
[root@VM_52_146_centos home]# tar xzf redis-3.2.8.tar.gz
[root@VM_52_146_centos home]# cd redis-3.2.8
[root@VM_52_146_centos redis-3.2.8]# make

`</pre>

[![](http://www.sevenyuan.cn/wp-content/uploads/2017/05/Reids解压成功.png)](http://www.sevenyuan.cn/wp-content/uploads/2017/05/Reids解压成功.png)

安装好之后进行基本配置

<pre>`[root@VM_52_146_centos redis-3.2.8]# ./utils/install_server.sh
`</pre>

我是按照默认进行配置的,所以显示一个就回车确认一个.

[![](http://www.sevenyuan.cn/wp-content/uploads/2017/05/进行配置.png)](http://www.sevenyuan.cn/wp-content/uploads/2017/05/进行配置.png)

最后使用redis-cli进入.

<pre>`[root@VM_52_146_centos redis-3.2.8]# redis-cli
127.0.0.1:6379&gt; ping
PONG
127.0.0.1:6379&gt;

这样就能进行基本的使用,具体更多使用和配置文件修改还需更多学习.
