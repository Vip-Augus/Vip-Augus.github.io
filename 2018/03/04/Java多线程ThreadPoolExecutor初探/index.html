JavaеӨҡзәҝзЁӢThreadPoolExecutorеҲқжҺў

еҸ‘иЎЁдәҺ 2018-03-04 | еҲҶзұ»дәҺ Java | жёёи§Ҳ ж¬Ў

еңЁjavaдёӯпјҢдҪҝз”ЁзәҝзЁӢж—¶йҖҡиҝҮnew Threadе®һзҺ°еҫҲз®ҖеҚ•пјҢдҪҶжҳҜеҰӮжһңе№¶еҸ‘ж•°йҮҸеҫҲеӨҡж—¶пјҢйў‘з№Ғең°еҲӣе»әзәҝзЁӢе°ұдјҡеӨ§еӨ§йҷҚдҪҺзі»з»ҹзҡ„ж•ҲзҺҮгҖӮ

жүҖд»ҘеҸҜд»ҘйҖҡиҝҮзәҝзЁӢжұ пјҢдҪҝеҫ—зәҝзЁӢеҸҜд»ҘеӨҚз”ЁпјҢжҜҸжү§иЎҢе®ҢдёҖдёӘд»»еҠЎпјҢе№¶дёҚжҳҜиў«й”ҖжҜҒпјҢиҖҢжҳҜеҸҜд»Ҙз»§з»ӯжү§иЎҢе…¶д»–д»»еҠЎгҖӮ

иҠұдәҶдёӨеӨ©ж—¶й—ҙеҺ»зңӢдәҶй«ҳжҙӘеІ©еҶҷзҡ„гҖҠJAVAе№¶еҸ‘зј–зЁӢгҖӢпјҢжҳҜжғіиҰҒзҹҘе…¶з„¶пјҢзҹҘе…¶жүҖд»Ҙз„¶пјҢеңЁдҪҝз”Ёзҡ„жғ…еҶөдёӢпјҢдәҶи§ЈеӯҰд№ дәҶдёҖдёӢеҺҹзҗҶи®°еҪ•дёӢjava.util.concurrentе№¶еҸ‘еҢ…дёӢзҡ„ThreadPoolExecutorзү№жҖ§е’Ңе®һзҺ°

дҪҝз”ЁзӨәдҫӢ

зІ—жҡҙзӮ№пјҢжҲ‘д»¬зӣҙжҺҘзңӢеҰӮдҪ•дҪҝз”Ёеҗ§

(дёҖ)дҪҝз”ЁExecutors

йҖҡиҝҮиҜҘExecutorsзҡ„йқҷжҖҒж–№жі•иҝӣиЎҢзәҝзЁӢжұ зҡ„еҲӣе»әпјҢиҖҢдё”д»Һе…·дҪ“е®һзҺ°жқҘзңӢпјҢиҝҳжҳҜи°ғз”ЁдәҶnew ThreadPoolExecutor()пјҢеҸӘжҳҜеҶ…йғЁеҸӮж•°е·Із»Ҹеё®жҲ‘д»¬й…ҚзҪ®еҘҪдәҶгҖӮ

(дәҢ) дҪҝз”ЁThreadPoolExecutor

ж—ўз„¶зңҹжӯЈе®һзҺ°йғҪжҳҜз”ЁThreadPoolExecutorпјҢйӮЈе°ұиҮӘе·ұи®ҫе®ҡеҘҪж–№жі•зҡ„еҸӮж•°еҗ§гҖӮ

жү“еҚ°ж•ҲжһңеҰӮдёӢпјҡ

д»»еҠЎTaskжҸҗдәӨд№ӢеҗҺпјҢз”ұдәҺжҳҜеӨҡзәҝзЁӢзҠ¶жҖҒдёӢпјҢжүҖд»Ҙжү“еҚ°ж•Ҳжһңе№¶дёҚжҳҜеҗҢжӯҘзҡ„пјҢеҸҜд»ҘзңӢеҮәд»»еҠЎйғҪе·Із»ҸйЎәеҲ©жү§иЎҢгҖӮ

жҲ‘иҝҷдёӘе®һзҺ°еҸӮж•°жҳҜ5дёӘcorePoolSizeж ёеҝғзәҝзЁӢж•°е’Ң5дёӘmaximumPoolSizeжңҖеӨ§зәҝзЁӢж•°пјҢеҪ“зәҝзЁӢжұ дёӯзҡ„зәҝзЁӢж•°и¶…иҝҮ5дёӘзҡ„ж—¶еҖҷпјҢе°Ҷж–°жқҘзҡ„д»»еҠЎж”ҫиҝӣзј“еӯҳйҳҹеҲ—дёӯпјҢе°ҸдјҷдјҙеҸҜд»ҘиҜ•дёӢжҠҠд»»еҠЎж•°(forеҫӘзҺҜзҡ„дёӘж•°)жҸҗй«ҳдёҖзӮ№пјҢи®©зј“еӯҳзӯүеҫ…зҡ„д»»еҠЎж•°и¶…иҝҮ5дёӘпјҢзңӢзңӢй»ҳи®Өзҡ„д»»еҠЎжӢ’з»қзӯ–з•Ҙ(AbortPolicy)дјҡжҠӣеҮәд»Җд№Ҳй”ҷиҜҜhhh

дёӢйқўжқҘзңӢзңӢThreadPoolExecutorзҡ„еәҗеұұзңҹйқўзӣ®еҗ§~

ThreadPoolExecutor

е®ғжңүд»ҘдёӢеӣӣдёӘжһ„йҖ ж–№жі•пјҡ

д»Һжһ„йҖ ж–№жі•еҸҜд»ҘзңӢеҮәпјҢеүҚдёүдёӘж–№жі•жңҖз»ҲйғҪжҳҜи°ғз”Ёз¬¬еӣӣдёӘжһ„йҖ еҷЁиҝӣиЎҢеҲқе§ӢеҢ–е·ҘдҪңзҡ„гҖӮ

еҸӮж•°и§ЈйҮҠпјҡ

  • corePoolSize:жұ дёӯдҝқжҢҒзҡ„зәҝзЁӢж•°пјҢеҢ…жӢ¬з©әй—Ізҡ„зәҝзЁӢпјҢд№ҹе°ұжҳҜж ёеҝғжұ зҡ„еӨ§е°Ҹ
  • maximumPoolSize:жұ дёӯй”Ғе…Ғи®ёжңҖеӨ§зәҝзЁӢж•°
  • keepAliveTime:еҪ“зәҝзЁӢж•°йҮҸи¶…иҝҮcorePoolSizeпјҢеңЁжІЎжңүи¶…иҝҮжҢҮе®ҡзҡ„ж—¶й—ҙеҶ…дёҚд»ҺзәҝзЁӢжұ дёӯеҲ йҷӨпјҢеҰӮжһңи¶…иҝҮиҜҘж—¶й—ҙпјҢеҲҷеҲ йҷӨ
  • unit:keepAliveTimeзҡ„ж—¶й—ҙеҚ•дҪҚ
  • workQueue:жү§иЎҢеүҚз”ЁжқҘдҝқеӯҳд»»еҠЎзҡ„йҳҹеҲ—пјҢжӯӨйҳҹеҲ—еҸӘдҝқеӯҳз”ұexecuteж–№жі•жҸҗдәӨзҡ„Runnableд»»еҠЎ

workQueueпјҲд»»еҠЎйҳҹеҲ—пјҢжҳҜдёҖдёӘйҳ»еЎһйҳҹеҲ—пјү

LinkedBlockingDeque:пјҲж”ҜжҢҒеҲ—еӨҙе’ҢеҲ—е°ҫж“ҚдҪңпјҢpollFirst/pollLast)

д»Һжәҗз Ғжһ„йҖ еҮҪж•°еҸҜд»ҘзңӢеҲ°пјҢдёҚдј еҸӮж•°зҡ„ж—¶еҖҷпјҢй»ҳи®Өйҳ»еЎһйҳҹеҲ—дёӯзҡ„еӨ§е°ҸжҳҜInteger.MAX_VALUE;

SynchronousQueue:

Arrayе’ҢLinkedеңЁдј е…ҘеӨ§е°Ҹе°ҸдәҺ0ж—¶е°ҶдјҡжҠҘй”ҷпјҢжҜ”иҫғеёёз”Ёзҡ„жҳҜLinkedBlockingDequeе’ҢSynchronousQueueпјҢзәҝзЁӢжұ зҡ„жҺ’йҳҹзӯ–з•ҘдёҺBlockingQueueжңүе…ігҖӮ

ThreadFactoryпјҡзәҝзЁӢе·ҘеҺӮ

дё»иҰҒз”ЁжқҘеҲӣе»әзәҝзЁӢпјҢеҸҜд»ҘеңЁnewThread()ж–№жі•дёӯиҮӘе®ҡд№үзәҝзЁӢеҗҚеӯ—е’Ңи®ҫзҪ®зәҝзЁӢејӮеёёжғ…еҶөзҡ„еӨ„зҗҶйҖ»иҫ‘гҖӮ

дёҫдёӘрҹҢ°пјҡ

жңүд»ҘдёӢеӣӣз§Қ:

  • ThreadPoolExecutor.AbortPolicyпјҡеҪ“д»»еҠЎж·»еҠ еҲ°зәҝзЁӢдёӯиў«жӢ’з»қж—¶пјҢе®ғдјҡжҠӣеҮәRejectedExecutionExceptionејӮеёёгҖӮ
  • ThreadPoolExecutor.DiscardPolicyпјҡд»»еҠЎиў«жӢ’з»қж—¶пјҢзәҝзЁӢжұ дёўејғиў«жӢ’з»қзҡ„д»»еҠЎ
  • ThreadPoolExecutor.DiscardOldestPolicyпјҡд»»еҠЎиў«жӢ’з»қж—¶пјҢзәҝзЁӢжұ дјҡж”ҫејғзӯүеҫ…йҳҹеҲ—дёӯжңҖж—§зҡ„жңӘеӨ„зҗҶж–Үзү©пјҢз„¶еҗҺе°Ҷиў«жӢ’з»қзҡ„д»»еҠЎж·»еҠ еҲ°зӯүеҫ…йҳҹеҲ—дёӯ
  • ThreadPoolExecutor.CallerRunsPolicyпјҡд»»еҠЎиў«жӢ’з»қж—¶пјҢдјҡдҪҝз”Ёи°ғз”ЁзәҝзЁӢжұ зҡ„ThreadзәҝзЁӢеҜ№иұЎеӨ„зҗҶиў«жӢ’з»қзҡ„д»»еҠЎ

ThreadPoolExecutorз»§жүҝз»“жһ„

еҸҜд»ҘзңӢеҮәпјҢе®һйҷ…дёҠThreadPoolExecutorжҳҜз»§жүҝдәҶAbstractExecutorServiceзұ»е’Ңеј•з”ЁдәҶExecutorServiceгҖҒExecutorжҺҘеҸЈгҖӮ

AbstractExecutorService

AbstarctExecutorServiceжҳҜдёҖдёӘжҠҪиұЎзұ»пјҢе®ғе®һзҺ°зҡ„жҳҜExecutorServiceжҺҘеҸЈ

ExecutorService

жҺҘеҸЈExecutorServiceеј•з”ЁдәҶExecutorжҺҘеҸЈпјҢExecutorжҺҘеҸЈжҜ”иҫғз®ҖеҚ•пјҢеҸӘжңүдёҖдёӘexecuteж–№жі•е®ҡд№ү

Executor

е°Ҹз»“пјҡ

ExecutorжҳҜдёҖдёӘйЎ¶зә§жҺҘеҸЈпјҢе®ҡд№үдәҶдёҖдёӘexecuteж–№жі•пјҢиҝ”еӣһеҖјдёәз©әпјҢеҸӮж•°дёәRunnableгҖӮ

ExecutorServiceз»§жүҝдәҶExecutorе№¶дё”е®ҡд№үдәҶе…¶е®ғдёҖдәӣж–№жі•пјҢз»“жһңеҰӮдёӢеӣҫпјҡ

жҠҪиұЎзұ»AbstractExecutorServiceе®һзҺ°дәҶExecutorServiceжҺҘеҸЈпјҢеҹәжң¬е®һзҺ°дәҶExecutorServiceдёӯеЈ°жҳҺзҡ„жүҖжңүж–№жі•гҖӮ

жңҖеҗҺThreadPoolExecutorз»§жүҝдәҶAbstractExecutorServiceпјҢжҲ‘д»¬жңҖеёёз”ЁеҲ°е®ғдёӨдёӘж–№жі•пјҢsubmitе’ҢexecuteпјҢдёӢйқўд»Ӣз»ҚдёҖдёӢиҝҷдёӨиҖ…пјҡ

е°Ҹз»“

execute()ж–№жі•еңЁThreadPoolExecutorдёӯиҝӣиЎҢдәҶйҮҚеҶҷпјҢsubmit()ж–№жі•жҳҜеңЁAbstractExecutorServiceе®һзҺ°зҡ„пјҢThreadPoolExecutorе№¶жІЎжңүйҮҚеҶҷпјҢе№¶дё”executeж–№жі•жҳҜжІЎжңүиҝ”еӣһз»“жһңзҡ„пјҢsubmitзҡ„иҝ”еӣһзұ»еһӢжҳҜFutureпјҢиғҪеӨҹиҺ·еҫ—д»»еҠЎзҡ„з»“жһңпјҢдҪҶжҳҜе®һйҷ…жү§иЎҢзҡ„иҝҳжҳҜexecuteж–№жі•гҖӮ

еҪ“з„¶пјҢиҝҳжңүдҫӢеҰӮshutdownгҖҒgetQueueгҖҒgetActiveCountгҖҒgetPoolSizeзӯүж–№жі•жІЎжңүд»Ӣз»ҚеҲ°пјҢжҺЁиҚҗиғ–еҸӢд»¬жү“ејҖIDEиҝӣиЎҢжҹҘзңӢеҗ§~

psпјҡе…ідәҺзәҝзЁӢжұ зҡ„еҺҹзҗҶе№¶жңӘж·ұе…Ҙи®°еҪ•пјҢжңүе…іе®ғзҡ„д»»еҠЎжӢ’з»қзӯ–з•ҘгҖҒзәҝзЁӢеҲқе§ӢеҢ–гҖҒThreadPoolExecutorжһ„йҖ д№ӢеҗҺпјҢеҪ“д»»еҠЎи¶…иҝҮи®ҫе®ҡеҖјпјҢе®ғзҡ„жү§иЎҢзӯ–з•ҘзӯүеҺҹзҗҶйғҪеҖјеҫ—еҺ»ж·ұе…ҘеӯҰд№ пјҢдёӢеӣһи®°еҪ•~
